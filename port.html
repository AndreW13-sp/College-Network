Web servers

A Web server is a specific computer, in which carries servers are where websites are engraved in HTML ( Hyper text mark-up language). The use for a web server is when a user tries to enter a website using a URL address such as www.bing.co.uk. When it works the server will then bring up the website that the user requested. The use of web servers is so that every computer is able to retrieve any web pages, the computer is connected to the server by the internet connection and that will allow the internet to be flown to the computer when you type in a URL address. So when someone published a web site it will be send to the server allowing every computer to access that website.
